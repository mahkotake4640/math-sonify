## 📖 What is math-sonify?

math-sonify turns math equations into sound. It uses well-known mathematical systems like Lorenz, Rössler, and Kuramoto to create real-time audio. These sounds come from complex shapes and chaos patterns, but you don’t need to understand the math behind them. The app maps these patterns directly to music, producing unique ambient and generative audio.

You will hear the behavior of dynamic systems as you interact with the software. It’s a simple tool to explore math and sound together.

## 🎛 How to Use math-sonify

Once open, math-sonify will immediately begin generating sound based on mathematical models. The interface shows controls and options with clear labels.

---

### Choose a System

You can select from several systems to change the sound:

- **Lorenz**: Chaotic weather model, creating swirling sounds
- **Rössler**: Another chaotic pattern with smooth oscillations
- **Kuramoto**: Synchronization of oscillators, producing rhythmic tones
- **Three-body**: Complex space object motion, creating irregular beats
- **Geodesic flows**: Patterns on curved spaces, adding texture to sound

To select a system, click the dropdown menu and pick one.

---

### Adjust Parameters

Each system has sliders or input fields to control its settings. Changes affect the sound in real time. For example, you can modify:

- Speed
- Volume levels
- Frequency ranges
- Noise intensity

Try adjusting the sliders to see how the sound changes.

---

### Pause or Stop

Use the pause button to stop the sound without closing the app. Hit "Stop" to reset the parameters to default.
